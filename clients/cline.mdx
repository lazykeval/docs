This guide will walk you through integrating Cline with ToolRouter to enable tool use capabilities.

## Setup

Create a ToolRouter Stack -> Create SSE Client -> Copy Cline Implementation code

<img src="/images/integration/cline/0.png" alt="Cline 1" />

Go to Cline Settings and click on "Remote Servers"

<img src="/images/integration/cline/1.png" alt="Cline 2" />

Add Server Name & SSE URL -> Click on "Add Server"

<img src="/images/integration/cline/2.png" alt="Cline 3" />

Cline might take a minute or two to connect to the server and add your tools.

<img src="/images/integration/cline/3.png" alt="Cline 4" />

Now, you can use Cline Chat with all the ToolRouter tools.

<img src="/images/integration/cline/4.png" alt="Cline 5" />
