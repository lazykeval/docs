This guide will walk you through integrating Windsurf with ToolRouter to enable tool use capabilities.

## Setup

Create a ToolRouter Stack -> Create SSE Client -> Copy Windsurf Implementation code

<img src="/images/integration/windsurf/0.png" alt="Windsurf 1" />

Go to Windsurf and select "Configure"

<img src="/images/integration/windsurf/1.png" alt="Windsurf 2" />

Paste the copied implementation code into the `mcp_config.json` file. Hit "Refresh" next to the "Configure" button.

<img src="/images/integration/windsurf/2.png" alt="Windsurf 3" />

Windsurf will show all added tools within the MCP server.

<img src="/images/integration/windsurf/3.png" alt="Windsurf 4" />

Now, you can use Windsurf Chat with all the ToolRouter tools.
